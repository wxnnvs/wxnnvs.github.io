Can this be detected?

Short answer: Probably not.

Longer answer: Depends. Your SafeExamBrowser installation cannot and will not detect any modification (nor lock you out > See issue #2 ↗) when using this tool. However, there are external programs that your school or university may use to compare your program files with the original, and these may be able to detect a modified build. (Ex.: seb-win-verificator ↗)

Installation

For this to work you need write access to C:\Program Files\

Un-SEB is developed for SEB 3.9.0, older versions may work, but are not officially supported.

  1. Download patch-seb.exe from GitHub ↗
  2. Run the program as Administrator.
Manual installation (not recommended)

Only use this method if you cannot run the exe or want to install an older release.

  1. Choose a release you want to install.
  2. Download all dll files (SafeExamBrowser.*.dll).
  3. Navigate to C:\Program Files\SafeExamBrowser\Application\.
  4. Paste all downloaded dll's in this folder (overwrite the original ones).
